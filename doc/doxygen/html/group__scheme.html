Detailed Description

The tcm daemon application iswidely based on the integrated scheme script interpreter TinyScheme.

The TinyScheme implementation itself is integrated in form of a dynamic link library, runs in a background thread and offers an optional REPL. The corresponding implementation can be found in ucm_scheme.c.

The access to the communication channels is realized with native scheme functions implemented in C which can be found within the source file tcm_scheme_ext.c. The function init_tcm_ff registered natively implemented scheme functions via TinyScheme's foreign function interface. Refer to README for more detailed information about these functions.
